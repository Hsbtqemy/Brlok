# Brlok

Application d'entraînement bloc et pan - génération de séances.

## Installation

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
pip install -e ".[dev]"
```

Pour exécuter les tests : `pytest`

## Usage

- **GUI** : `python -m brlok`
- **CLI** : `brlok generate --help`
