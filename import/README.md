# Import automatique Brlok

Déposez ici des fichiers JSON au démarrage de l'app : Brlok les importe et les fusionne avec vos données, puis les déplace dans `imported/`.

| Fichier à déposer | Fusion |
|-------------------|--------|
| `favorites.json` | Blocs ajoutés aux favoris du catalogue actif |
| `templates.json` | Templates ajoutés (sans doublon de nom) |
| `catalog_collection.json` | Chaque catalogue ajouté avec préfixe « Importé - » |
